# AgentCollect Technical Assessment

This repository contains my written approach to investigating and debugging a UI event that is not behaving as expected.

Rather than focusing on a specific framework or implementation, I structured the document around how I would approach the problem in a real engineering environment:

* Reproducing the issue consistently
* Forming and validating hypotheses
* Isolating the root cause
* Verifying the fix
* Considering edge cases and regressions

The objective was to demonstrate a systematic debugging process and engineering reasoning rather than simply proposing a solution.
